# Product

## Register

brand

## Users

Privacy-conscious individuals who want to remove metadata from images before sharing online. Use cases: removing GPS location from photos, stripping AI generator signatures (Stable Diffusion, Midjourney, DALL-E), clearing C2PA content credentials, erasing device fingerprints. Typical context: desktop browser, one-time or occasional use, moderate technical literacy.

## Product Purpose

CleanMetaDATA strips EXIF, GPS, XMP, IPTC, and C2PA metadata from images entirely in the browser — no upload, no account, no server. Success looks like: user drops a photo, sees exactly what data was found and removed, downloads a clean file, feels confident their privacy is protected.

## Brand Personality

Trustworthy, Simple, Fast. The interface should feel like a precision tool — not clinical, not flashy. Users should trust it immediately and not need to read instructions.

## Anti-references

No generic SaaS patterns: no hero metrics (images cleaned! users! countries!), no identical card grid sections, no startup-green CTAs, no glassmorphism, no gradient text. No bloated enterprise aesthetic.

## Design Principles

1. **Show, don't describe** — surface the actual metadata found, not vague promises about privacy protection
2. **One task, no distractions** — every element on the page serves the upload-inspect-download flow
3. **Privacy by design** — the UI itself demonstrates the product's values: minimal, no trackers, no dark patterns
4. **Confidence through transparency** — users see exactly what was removed; no black-box "cleaning"
5. **Fast first** — tool feels instant; no loading skeletons where real content can load immediately

## Accessibility & Inclusion

WCAG 2.1 AA minimum. Dark theme default — ensure 4.5:1 contrast on all body text. Keyboard-navigable file upload. No motion-dependent affordances.
